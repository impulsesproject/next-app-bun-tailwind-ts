# Next.app.bun.ts

Impulse (Docker dev container configuration and scaffolding) for an app-router based [Next.js](https://nextjs.org/) 14 application built with [Bun](https://bun.sh/) and TypeScript support.

## Usage

Inside the container,

```bash
 # Running your app
 bun run dev

 # Building your app
 bun run build

 # Linting
 bun run lint

 # Formatting
 bun run format
```
